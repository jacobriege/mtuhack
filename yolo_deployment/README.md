# YOLO-Deployment

Hier geht es um das Deployment des YOLO-Modells bzw. der YOLO-Modelle. Deshalb wurden Skripte für das Deployment als auch Python-Files, die als Funktions-Bibliothek dienen, geschrieben.
<br> <br>
Des Weiteren wurden Tests gemacht, die `.pt`-Modell-Dateien in `ncnn`-Modelle zu konvertieren, die auf ARM-Prozessoren schneller laufen.
